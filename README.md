Este proyecto es una aplicación en Java que permite calcular el capital final aplicando interés compuesto, validando previamente los datos ingresados por el usuario.
El programa utiliza entrada por consola e incluye comprobaciones básicas para evitar valores inválidos.

Para ejecutar el programa puedes o bien ejecutarlo desde la terminal o clonarlo y usando VS Code, IntelliJ etc...
